# Digital-Clock-Arduino
Digital Clock with Arduino, displays real time, date, day of the week, temperature and humidity.

# Components Needed:
DHT11 Humidity and Temperature sensor
Arduino UNO
RTC DS1302 Real time clock
LDC screen 16x2 I2C
